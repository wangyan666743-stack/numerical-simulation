# Data Dictionary

Record field names, units, coordinate systems, and calculation methods here.

## Common Fields

| Field | Unit | Meaning | Source |
| --- | --- | --- | --- |
| run_id | - | Unique simulation run identifier | `metadata/run_log.csv` |
| date | YYYY-MM-DD | Simulation completion or archive date | Manual record |
| case_name | - | Short case name | Manual record |
| analysis_status | - | `raw`, `processed`, `analyzed`, or `verified` | Manual record |
